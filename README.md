# AOSP2

[![CI – AvfTerminal](https://github.com/kimocoder/AOSP2/actions/workflows/ci.yml/badge.svg)](https://github.com/kimocoder/AOSP2/actions/workflows/ci.yml)

This repository contains the **AvfTerminal** app — a repackaged import of the AVF Terminal app
from AOSP (`platform/packages/modules/Virtualization/android/TerminalApp/`) with the custom
package name `com.kimocoder.avfterminal`.

See [`packages/apps/AvfTerminal/README.md`](packages/apps/AvfTerminal/README.md) for details on
the import, build requirements, and known limitations.
